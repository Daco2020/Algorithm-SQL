Complete the function which returns the weekday according to the input number:

- 1 returns "Sunday"
- 2 returns "Monday"
- 3 returns "Tuesday"
- 4 returns "Wednesday"
- 5 returns "Thursday"
- 6 returns "Friday"
- 7 returns "Saturday"
- Otherwise returns "Wrong, please enter a number between 1 and 7"

<br><br>

## Solution:
```py
days = {
    1: "Sunday",
    2: "Monday",
    3: "Tuesday",
    4: "Wednesday",
    5: "Thursday",
    6: "Friday",
    7: "Saturday"
}

def whatday(num: int) -> str:
    try:
        return days[num]
    except KeyError:
        return "Wrong, please enter a number between 1 and 7"
```
