Write a function that returns a string in which firstname is swapped with last name.

Example(Input --> Output)

```
"john McClane" --> "McClane john"
```

<br><br>

## Solution:
```py
def name_shuffler(str_):
    return " ".join(str_.split(" ")[::-1])
```
